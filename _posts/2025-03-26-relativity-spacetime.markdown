## Project Overview

This project visualizes key concepts from Einstein's theory of relativity:

- Time dilation effects
- Length contraction  
- Spacetime curvature

## Key Features

1. Interactive spacetime diagrams
2. Real-time relativity calculations
3. Educational explanations
